# <img src="appicon.png" align="left" width="96" height="96"/> OneWinDesk

一个让桌面只能单窗口显示的小工具
 


## 简介
项目基于 Win32 C++ 开发, 体积极小, 占用内存也低. 软件没有主界面GUI, 只有托盘图标和托盘菜单.

## 功能
- 桌面只能显示一个软件窗口, 打开新窗口时自动最小化旧窗口
- 开机自启动
- 将前台窗口进程添加进白名单
- 白名单管理
